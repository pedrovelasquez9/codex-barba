En este video te explico todo lo que necesitas saber **antes de comenzar con Angular**. Si te has preguntado:

> "¿Es mejor aprender primero JavaScript o TypeScript?"
> "¿Necesito dominar RxJS para empezar?"
> "¿HTML y CSS siguen siendo importantes?"

La respuesta corta es: **sí, necesitas una base clara**. La respuesta larga te la explico en el video y la complemento aquí.

## ✅ Checklist previa para aprender Angular

### 🟡 HTML y CSS

- Estructura semántica
- Uso de clases, selectores, flexbox y grid
- Media queries y responsive design

### 🟨 JavaScript moderno

- Sintaxis ES6+ (`let`, `const`, arrow functions)
- Módulos (`import`, `export`)
- Tipos de datos y estructuras
- Funciones, callbacks, asincronía (`Promise`)
- Manejo de errores

### 🔵 TypeScript

- POO básica (clases, interfaces)
- Tipos primitivos y avanzados
- Clases, constructores, herencia
- Utility types (`Partial`, `Pick`, etc.)

### 🟢 Conceptos adicionales

- SPA y renderizado
- RxJS y programación reactiva
- NPM y gestión de dependencias
- Debugging (DevTools, errores comunes)
