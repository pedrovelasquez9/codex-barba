# Introducción a Java y Programación Orientada a Objetos

Java es un lenguaje fuertemente tipado, compilado y orientado a objetos. En este vídeo exploramos sus fundamentos y cómo aplicar los principios básicos de la programación orientada a objetos.

## ¿Cómo funciona?

Java utiliza una máquina virtual para ejecutar su bytecode, lo que permite portabilidad entre sistemas. La clase principal debe tener un método `main` como punto de entrada, y la sintaxis es estricta respecto a tipos y estructuras.

## ¿Qué cubre este vídeo?

- Instalación y configuración del entorno de desarrollo con Java 21 e IntelliJ IDEA community edition.
- Tipos de datos primitivos y no primitivos en Java.
- Clases, objetos, atributos, métodos y constructores.
- Encapsulamiento, herencia, polimorfismo y abstracción.
- Diferencias entre clases abstractas e interfaces.
- Uso de `try`, `catch` y `finally` para manejo de errores.
- Ejemplos prácticos para ilustrar cada concepto.
